Điều kiện và Vòng lặp trong JavaScript

Mở đầu

Trong lập trình, câu lệnh điều kiện và vòng lặp là hai công cụ quan trọng giúp chương trình hoạt động theo logic mong muốn.

  • Câu lệnh điều kiện: cho phép chương trình ra quyết định dựa trên các điều kiện.
  • Vòng lặp: cho phép lặp lại một đoạn code nhiều lần, tiết kiệm thời gian và tránh lặp lại không cần thiết.

1. Câu lệnh điều kiện trong JavaScript

1.1. if

Dùng để kiểm tra một điều kiện, nếu điều kiện đúng (true) thì thực thi đoạn code.

let age = 20;

if (age >= 18) {
  console.log("Bạn đã đủ tuổi trưởng thành");
}

1.2. if...else

Thêm lựa chọn khi điều kiện sai (false).

let score = 45;

if (score >= 50) {
  console.log("Bạn đã đậu");
} else {
  console.log("Bạn đã rớt");
}

1.3. if...else if...else

Dùng khi có nhiều điều kiện cần kiểm tra.

let mark = 85;

if (mark >= 90) {
  console.log("Xếp loại A");
} else if (mark >= 75) {
  console.log("Xếp loại B");
} else if (mark >= 50) {
  console.log("Xếp loại C");
} else {
  console.log("Xếp loại D");
}

1.4. switch...case

Dùng khi có nhiều trường hợp cần so sánh, giúp code gọn gàng hơn.

let day = 3;

switch (day) {
  case 1:
    console.log("Thứ Hai");
    break;
  case 2:
    console.log("Thứ Ba");
    break;
  case 3:
    console.log("Thứ Tư");
    break;
  default:
    console.log("Không rõ ngày");
}

2. Vòng lặp trong JavaScript

2.1. for

Vòng lặp phổ biến, dùng khi biết trước số lần lặp.

for (let i = 1; i <= 5; i++) {
  console.log("Lần lặp:", i);
}

2.2. while

Lặp khi điều kiện còn đúng.

let i = 1;
while (i <= 5) {
  console.log("Lặp while:", i);
  i++;
}

2.3. do...while

Luôn chạy ít nhất 1 lần, rồi kiểm tra điều kiện.

let j = 1;
do {
  console.log("Lặp do...while:", j);
  j++;
} while (j <= 5);

2.4. for...of

Dùng để duyệt qua giá trị của mảng hoặc chuỗi.

let numbers = [10, 20, 30];

for (let num of numbers) {
  console.log("Giá trị:", num);
}

2.5. for...in

Dùng để duyệt qua key (chỉ mục hoặc thuộc tính) của object hoặc mảng.

let person = { name: "Kiệt", age: 21 };

for (let key in person) {
  console.log(key, ":", person[key]);
}

3. Từ khóa điều khiển vòng lặp

  • break: thoát khỏi vòng lặp ngay lập tức.
  • continue: bỏ qua vòng lặp hiện tại, nhảy sang vòng lặp kế tiếp.

Ví dụ:

for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    continue; // bỏ qua số 3
  }
  if (i === 5) {
    break; // dừng vòng lặp khi i = 5
  }
  console.log("Giá trị:", i);
}

Ví dụ tổng hợp

// Kiểm tra số chẵn lẻ trong khoảng 1-10
for (let i = 1; i <= 10; i++) {
  if (i % 2 === 0) {
    console.log(i, "là số chẵn");
  } else {
    console.log(i, "là số lẻ");
  }
}

Kết luận

  • Điều kiện (if, if...else, switch) giúp chương trình đưa ra quyết định.
  • Vòng lặp (for, while, do...while, for...of, for...in) giúp lặp lại công việc tự động.
  • Kết hợp điều kiện và vòng lặp giúp bạn xây dựng chương trình mạnh mẽ, logic và hiệu quả hơn.

Đây là nền tảng cơ bản để tiến tới các khái niệm nâng cao hơn trong JavaScript như hàm, mảng, object, class và lập trình bất đồng bộ.
