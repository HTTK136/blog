Biến và Kiểu dữ liệu trong JavaScript

Mở đầu

JavaScript là một trong những ngôn ngữ lập trình phổ biến nhất hiện nay, được dùng rộng rãi để xây dựng website, ứng dụng và nhiều công nghệ web hiện đại.
Để học tốt JavaScript, bạn cần nắm vững hai khái niệm quan trọng nhất: Biến (Variables) và Kiểu dữ liệu (Data Types).
Biến giúp bạn lưu trữ thông tin, còn kiểu dữ liệu quyết định dạng của thông tin đó. Hiểu rõ cách khai báo biến và các kiểu dữ liệu sẽ giúp bạn viết chương trình dễ bảo trì, dễ mở rộng và ít gặp lỗi hơn.

1. Biến trong JavaScript

Biến là “chiếc hộp” dùng để lưu trữ giá trị trong chương trình.
JavaScript hỗ trợ 3 cách khai báo biến: var, let và const.

1.1. var

  • Là từ khóa cũ, được giới thiệu từ phiên bản đầu tiên của JavaScript.
  • Phạm vi (scope): function scope (chỉ tồn tại trong hàm).
  • Có thể khai báo lại nhiều lần trong cùng phạm vi.
  • Dễ gây lỗi khi dùng trong vòng lặp hoặc trong các block code.

Ví dụ:

var x = 10;
var x = 20; // hợp lệ
console.log(x); // 20

1.2. let

  • Giới thiệu từ ES6 (2015), hiện là cách khai báo biến được khuyến khích sử dụng.
  • Phạm vi: block scope (chỉ tồn tại trong { ... }).
  • Không thể khai báo lại trong cùng một phạm vi.
  • Thường dùng cho những giá trị có thể thay đổi.

Ví dụ:

let y = 15;
y = 30; // hợp lệ
console.log(y); // 30

// let y = 40; ❌ báo lỗi nếu khai báo lại

1.3. const

  • Cũng xuất hiện từ ES6.
  • Dùng để khai báo hằng số, nghĩa là giá trị không thể thay đổi sau khi gán.
  • Phạm vi: block scope.
  • Nếu khai báo object hoặc array với const, bạn có thể thay đổi nội dung bên trong, nhưng không thể gán lại biến.

Ví dụ:

const PI = 3.14;
// PI = 4; ❌ báo lỗi

const person = { name: "Kiệt", age: 20 };
person.age = 21; // hợp lệ (thay đổi thuộc tính trong object)

👉 Tóm tắt nhanh:

Từ khóa Phạm vi Có thể gán lại Có thể khai báo lại Ghi chú
var function ✔ ✔ Ít dùng trong code hiện đại
let block ✔ ❌ Dùng cho biến có thể thay đổi
const block ❌ ❌ Dùng cho hằng số hoặc object/array không đổi tham chiếu

2. Kiểu dữ liệu trong JavaScript

JavaScript là ngôn ngữ dynamic typing (kiểu động) → khi khai báo biến, bạn không cần chỉ định kiểu dữ liệu.
Kiểu dữ liệu sẽ được xác định dựa vào giá trị được gán.

2.1. Kiểu dữ liệu nguyên thủy (Primitive)

Có 7 loại kiểu dữ liệu cơ bản:

  1. Number – dùng cho số nguyên và số thực.

    let a = 10;
let b = 3.14;

  2. String – chuỗi ký tự, viết bằng ' ', " " hoặc ` (template string).

    let name = "Kiệt";
let greeting = `Xin chào, ${name}`;

  3. Boolean – chỉ nhận true hoặc false.

    let isActive = true;
let isAdmin = false;

  4. Undefined – khi biến được khai báo nhưng chưa gán giá trị.

    let x;
console.log(x); // undefined

  5. Null – giá trị rỗng, thường dùng khi muốn reset biến.

    let y = null;

  6. Symbol (ES6) – tạo giá trị duy nhất, thường dùng để làm key cho object.

    let id = Symbol("id");

  7. BigInt (ES11) – dùng cho số nguyên rất lớn.

    let big = 123456789012345678901234567890n;

2.2. Kiểu dữ liệu tham chiếu (Reference)

Khác với kiểu nguyên thủy, kiểu tham chiếu lưu địa chỉ ô nhớ, không lưu giá trị trực tiếp.

  1. Object – tập hợp các cặp key–value.

    let person = {
  name: "Kiệt",
  age: 21
};

  2. Array – mảng, dùng để lưu danh sách nhiều giá trị.

    let numbers = [1, 2, 3, 4, 5];

  3. Function – là object đặc biệt có thể gọi thực thi.

    function sayHello() {
  console.log("Hello!");
}

3. Kiểm tra kiểu dữ liệu với typeof

JavaScript cung cấp toán tử typeof để kiểm tra kiểu dữ liệu:

console.log(typeof 10);          // "number"
console.log(typeof "abc");       // "string"
console.log(typeof true);        // "boolean"
console.log(typeof undefined);   // "undefined"
console.log(typeof null);        // "object" (điểm đặc biệt trong JS)
console.log(typeof {});          // "object"
console.log(typeof []);          // "object"
console.log(typeof function(){}); // "function"

Ví dụ tổng hợp

let age = 21;                 // Number
const fullName = "Huỳnh Kiệt"; // String
let isStudent = true;         // Boolean
let address;                  // Undefined
let car = null;               // Null
let skills = ["JS", "HTML"];  // Array
let profile = { name: "Kiệt", year: 2004 }; // Object

function showInfo() {
  console.log(`Tên: ${profile.name}, Năm sinh: ${profile.year}`);
}

Kết luận

  • Biến trong JavaScript có thể khai báo bằng var, let hoặc const. Trong đó, let và const là lựa chọn an toàn và hiện đại hơn.
  • Kiểu dữ liệu chia thành 2 nhóm:
    • Nguyên thủy: Number, String, Boolean, Undefined, Null, Symbol, BigInt.
    • Tham chiếu: Object, Array, Function.
  • JavaScript là ngôn ngữ động kiểu, vì vậy bạn cần cẩn thận khi làm việc với dữ liệu để tránh lỗi không mong muốn.

Hiểu rõ biến và kiểu dữ liệu là nền tảng để học các phần nâng cao hơn như hàm, đối tượng, class, hay lập trình bất đồng bộ. Đây chính là bước khởi đầu vững chắc để trở thành một lập trình viên JavaScript giỏi.
