JSON trong JavaScript

Mở đầu

JSON (JavaScript Object Notation) là một định dạng dữ liệu phổ biến, nhẹ và dễ đọc, thường được dùng để trao đổi dữ liệu giữa client (trình duyệt) và server.
Trong JavaScript, JSON rất quan trọng vì nó là cách phổ biến để lưu trữ và truyền dữ liệu trong các ứng dụng web hiện đại.

1. JSON là gì?

  • JSON có cấu trúc tương tự như object trong JavaScript.
  • Dữ liệu trong JSON được biểu diễn dưới dạng cặp key–value.
  • Giá trị có thể là: number, string, boolean, null, object, array.

Ví dụ JSON:

{
  "name": "Kiệt",
  "age": 21,
  "isStudent": true,
  "skills": ["JavaScript", "HTML", "CSS"],
  "address": {
    "city": "Hồ Chí Minh",
    "country": "Việt Nam"
  }
}

2. Sự khác biệt giữa JSON và Object trong JavaScript

  • JSON là một chuỗi (string) theo định dạng chuẩn.
  • Object là kiểu dữ liệu của JavaScript.

Ví dụ:

// Object trong JS
let person = {
  name: "Kiệt",
  age: 21
};

// JSON (string)
let jsonString = '{"name": "Kiệt", "age": 21}';

3. Làm việc với JSON trong JavaScript

3.1. JSON.stringify()

Chuyển Object → JSON string.

let obj = { name: "Kiệt", age: 21 };
let json = JSON.stringify(obj);

console.log(json);
// Kết quả: '{"name":"Kiệt","age":21}'

3.2. JSON.parse()

Chuyển JSON string → Object.

let jsonStr = '{"name":"Kiệt","age":21}';
let obj = JSON.parse(jsonStr);

console.log(obj.name); // "Kiệt"
console.log(obj.age);  // 21

4. JSON và API

Khi làm việc với API, dữ liệu thường được trả về dưới dạng JSON.

Ví dụ (giả sử gọi API):

// Giả lập dữ liệu trả về từ server
let response = '{"id":1,"title":"Học JSON","completed":false}';

// Parse JSON thành Object
let data = JSON.parse(response);

console.log(data.title); // "Học JSON"

Ngược lại, khi gửi dữ liệu lên server, bạn thường phải chuyển Object thành JSON:

let newUser = { name: "Lan", age: 20 };

let jsonData = JSON.stringify(newUser);
console.log(jsonData); // '{"name":"Lan","age":20}'

5. Ưu điểm của JSON

  • Dễ đọc và dễ hiểu cho con người.
  • Nhẹ và dễ phân tích.
  • Ngôn ngữ độc lập: không chỉ dùng trong JavaScript, mà còn trong Python, Java, PHP…
  • Tương thích với API: hầu hết các API hiện nay dùng JSON.

Ví dụ tổng hợp

// Object ban đầu
let product = {
  id: 101,
  name: "Laptop",
  price: 1500,
  inStock: true
};

// Chuyển thành JSON string
let jsonProduct = JSON.stringify(product);
console.log("JSON:", jsonProduct);

// Chuyển ngược lại thành Object
let objProduct = JSON.parse(jsonProduct);
console.log("Tên sản phẩm:", objProduct.name);

Kết quả:

JSON: {"id":101,"name":"Laptop","price":1500,"inStock":true}
Tên sản phẩm: Laptop

Kết luận

  • JSON là định dạng dữ liệu phổ biến trong trao đổi dữ liệu web.
  • Có 2 hàm quan trọng: JSON.stringify() (Object → JSON) và JSON.parse() (JSON → Object).
  • JSON là cầu nối quan trọng khi làm việc với API, server, database trong lập trình JavaScript.

Hiểu và làm chủ JSON sẽ giúp bạn làm việc hiệu quả hơn trong các dự án web hiện đại.
