Mảng (Array) trong JavaScript

Mở đầu

Trong lập trình, mảng (array) là cấu trúc dữ liệu quan trọng giúp lưu trữ nhiều giá trị trong một biến duy nhất.
Trong JavaScript, mảng rất linh hoạt: có thể chứa nhiều kiểu dữ liệu khác nhau (số, chuỗi, boolean, object, thậm chí cả hàm).

1. Khai báo mảng

Có nhiều cách để khai báo mảng trong JavaScript:

1.1. Dùng dấu [] (thường dùng nhất)

let numbers = [1, 2, 3, 4, 5];
let mixed = [1, "hello", true, null];

1.2. Dùng từ khóa new Array()

let fruits = new Array("Táo", "Cam", "Chuối");

2. Truy cập phần tử mảng

  • Các phần tử được đánh chỉ số (index) bắt đầu từ 0.
let fruits = ["Táo", "Cam", "Chuối"];
console.log(fruits[0]); // "Táo"
console.log(fruits[2]); // "Chuối"
  • Thay đổi giá trị phần tử:
fruits[1] = "Xoài";
console.log(fruits); // ["Táo", "Xoài", "Chuối"]

3. Thuộc tính và phương thức cơ bản của mảng

3.1. Thuộc tính length

Trả về số lượng phần tử trong mảng.

let nums = [10, 20, 30];
console.log(nums.length); // 3

3.2. Thêm / xóa phần tử

  • push(): thêm vào cuối mảng.
  • pop(): xóa phần tử cuối.
  • unshift(): thêm vào đầu mảng.
  • shift(): xóa phần tử đầu.
let animals = ["Mèo", "Chó"];
animals.push("Cá");   // ["Mèo", "Chó", "Cá"]
animals.pop();        // ["Mèo", "Chó"]
animals.unshift("Gà"); // ["Gà", "Mèo", "Chó"]
animals.shift();      // ["Mèo", "Chó"]

3.3. Kết hợp và cắt mảng

  • concat(): nối mảng.
  • slice(): cắt mảng.
  • splice(): thêm / xóa tại vị trí bất kỳ.
let a = [1, 2], b = [3, 4];
let c = a.concat(b); // [1, 2, 3, 4]

let nums = [10, 20, 30, 40];
console.log(nums.slice(1, 3)); // [20, 30]

nums.splice(2, 1, 99); // xóa 1 phần tử ở index=2 và thêm 99
console.log(nums); // [10, 20, 99, 40]

3.4. Tìm kiếm trong mảng

  • indexOf() → trả về vị trí đầu tiên tìm thấy.
  • includes() → kiểm tra có tồn tại không.
let colors = ["đỏ", "xanh", "vàng"];
console.log(colors.indexOf("xanh")); // 1
console.log(colors.includes("tím")); // false

4. Duyệt mảng

4.1. for

let arr = [1, 2, 3];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}

4.2. for...of

for (let value of arr) {
  console.log(value);
}

4.3. forEach()

arr.forEach((value, index) => {
  console.log("Index:", index, "Giá trị:", value);
});

5. Các phương thức xử lý mảng hiện đại

5.1. map() – tạo mảng mới sau khi biến đổi

let nums = [1, 2, 3];
let squares = nums.map(x => x * x);
console.log(squares); // [1, 4, 9]

5.2. filter() – lọc các phần tử thỏa điều kiện

let nums = [5, 10, 15, 20];
let big = nums.filter(x => x > 10);
console.log(big); // [15, 20]

5.3. reduce() – gộp mảng thành một giá trị duy nhất

let nums = [1, 2, 3, 4];
let sum = nums.reduce((acc, cur) => acc + cur, 0);
console.log(sum); // 10

5.4. find() và findIndex()

let nums = [5, 12, 8, 130];
console.log(nums.find(x => x > 10)); // 12
console.log(nums.findIndex(x => x > 10)); // 1

Ví dụ tổng hợp

let students = [
  { name: "Kiệt", score: 85 },
  { name: "Lan", score: 92 },
  { name: "Minh", score: 65 }
];

// Lấy danh sách học sinh đậu (score >= 70)
let passed = students.filter(s => s.score >= 70);

// In ra tên học sinh
passed.forEach(s => console.log(s.name));

Lan

Kết luận

  • Mảng là cấu trúc dữ liệu quan trọng để lưu danh sách giá trị.
  • JavaScript cung cấp rất nhiều phương thức mạnh mẽ để thao tác với mảng (map, filter, reduce, …).
  • Việc nắm chắc kiến thức về mảng sẽ giúp bạn làm việc hiệu quả hơn với dữ liệu, đặc biệt khi kết hợp với object và hàm bậc cao trong JavaScript.
