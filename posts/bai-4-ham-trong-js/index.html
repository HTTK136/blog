Hàm trong JavaScript

Mở đầu

Trong JavaScript, hàm (function) là khối mã (code block) có thể tái sử dụng nhiều lần để thực hiện một công việc cụ thể.
Hàm giúp chương trình dễ bảo trì, dễ đọc và giảm lặp code. Đây là một trong những khái niệm cốt lõi bạn phải nắm vững khi học JavaScript.

1. Định nghĩa hàm

Hàm trong JavaScript là một tập hợp các câu lệnh được nhóm lại, có thể gọi thực thi bất cứ khi nào cần.

Cú pháp cơ bản:

function tenHam(thamSo1, thamSo2, ...) {
  // khối lệnh
  return giaTri; // (tùy chọn)
}

Ví dụ:

function greet(name) {
  return "Xin chào, " + name;
}

console.log(greet("Kiệt")); // Kết quả: Xin chào, Kiệt

2. Khai báo hàm

2.1. Function Declaration (Hàm khai báo)

function sum(a, b) {
  return a + b;
}
console.log(sum(5, 7)); // 12

Đặc điểm: có thể gọi hàm trước khi khai báo (do hoisting).

console.log(add(3, 4)); // 7
function add(x, y) {
  return x + y;
}

2.2. Function Expression (Hàm biểu thức)

const multiply = function(a, b) {
  return a * b;
};
console.log(multiply(3, 4)); // 12

Đặc điểm: không được hoisting → phải khai báo trước khi gọi.

2.3. Arrow Function (Hàm mũi tên) – ES6

Cú pháp ngắn gọn hơn.

const divide = (a, b) => a / b;
console.log(divide(10, 2)); // 5

Nếu hàm chỉ có một tham số, có thể bỏ ngoặc:

const square = x => x * x;
console.log(square(5)); // 25

2.4. IIFE (Immediately Invoked Function Expression)

Hàm tự gọi ngay sau khi định nghĩa.

(function() {
  console.log("Đây là IIFE");
})();

3. Tham số và giá trị trả về

  • Tham số (parameters): giá trị đưa vào hàm khi khai báo.
  • Đối số (arguments): giá trị truyền vào hàm khi gọi.
  • Giá trị trả về (return): kết quả mà hàm trả ra.

Ví dụ:

function calculateArea(width, height) {
  return width * height;
}
console.log(calculateArea(5, 10)); // 50

Nếu không có return, hàm sẽ trả về undefined.

4. Hàm trong hàm (Nested Functions)

Có thể định nghĩa hàm bên trong một hàm khác.

function outer() {
  function inner() {
    console.log("Hàm bên trong");
  }
  inner();
}
outer();

5. Các tính năng đặc biệt của hàm trong JS

  • Hàm là một object: có thể gán vào biến, truyền làm tham số hoặc trả về từ hàm khác.
  • Closures: hàm con có thể truy cập biến trong phạm vi của hàm cha. 
    function makeCounter() {
  let count = 0;
  return function() {
    count++;
    return count;
  };
}

const counter = makeCounter();
console.log(counter()); // 1
console.log(counter()); // 2

Ví dụ tổng hợp

// Hàm kiểm tra số nguyên tố
function isPrime(n) {
  if (n < 2) return false;
  for (let i = 2; i <= Math.sqrt(n); i++) {
    if (n % i === 0) return false;
  }
  return true;
}

for (let i = 1; i <= 10; i++) {
  console.log(i, isPrime(i) ? "là số nguyên tố" : "không phải số nguyên tố");
}

Kết luận

  • Hàm giúp tái sử dụng code và tổ chức chương trình tốt hơn.
  • Có nhiều cách khai báo hàm: declaration, expression, arrow function, IIFE.
  • Hàm trong JS còn hỗ trợ closure và coi hàm như một object đặc biệt.

Việc hiểu rõ và sử dụng thành thạo hàm là bước quan trọng để học các khái niệm nâng cao như callback, higher-order functions, async/await.
